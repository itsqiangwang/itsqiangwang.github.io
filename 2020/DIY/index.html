D.I.Y
Select issues for D.I.Y, an independent publishing project that explores the book as a container for ideas. d.i.y stands for does it interest you and refers to the way these publications materialize (printed and assembled by the readers themselves) as well as the goal of this project (to serve as a container for my experimentations and explorations with ideas). all publications are designed in black and white to reduce printing cost and assembled using the slot-and-tab method, which only requires a ruler and a pair of scissors.
