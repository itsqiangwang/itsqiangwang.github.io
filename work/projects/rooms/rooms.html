A mixed media project created for my 'Drawing + Color' class during Winter 2017.

This goal of this project was to "explore the boundaries and in-between spaces of categories in art: from painting to sculpture, from objects to images, from stills to video."

Different representation techniques were used to create a mixed media piece that was later published as a website.
